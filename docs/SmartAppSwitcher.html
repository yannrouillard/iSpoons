docs » SmartAppSwitcher

Easily switch between your most used applications with one key assigned to each app, and automatically activate the most relevant window/tab in the target application when possible.

For instance, when switching from iTerm2 to Visual Studio Code while on a source code repository, SmartAppSwitcher will automatically activate or open a Visual Studio Code window on the source code repository.

Notes:

  • The window/tab auto-select feature requires some helper spoons for eligible applications (currently only iTerm2 and Visual Studio Code are supported)

Examples configuration:

hs.loadSpoon("SmartAppSwitcher")
spoon.SmartAppSwitcher.modifier = {"cmd", "alt"}
spoon.SmartAppSwitcher.helperSpoons = {
  ["iTerm"] = spoon.Iterm2,
  ["Visual Studio Code"] = spoon.VsCode
}
spoon.SmartAppSwitcher.bindHotkeys({
  c = "Visual Studio Code",
  o = "Obsidian",
  f = "Firefox",
  k = "Slack",
  s = "Spotify",
  i = "iTerm"
})
